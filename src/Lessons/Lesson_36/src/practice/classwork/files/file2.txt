Для создания ZIP архива нужно создать экземпляр ZipOutputStream,
передав в его конструктор объект FileOutputStream,
указывающий на файл, который будет создан как ZIP архив.
Затем, используя метод putNextEntry, можно добавлять в архив новые файлы,
а с помощью методов записи write добавлять содержимое файлов.
Keyword = "файл"
